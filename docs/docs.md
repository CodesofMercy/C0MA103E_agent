# C0MA103E Agent — Что это и зачем

## Идея

Агент — твой автономный менеджер музыкального проекта C0MA103E.
Он живёт на сервере, читает твои планы в Obsidian, напоминает что делать,
и помогает генерировать контент через AI.

## Что агент делает

**Каждое утро в 09:00** пишет тебе в Telegram:
- Что нужно сделать сегодня по контент-плану
- Что нужно подготовить заранее
- Что ближайшее из несделанного

**По команде `/plan`** — план на сегодня.
**По команде `/week`** — обзор задач на неделю.

**Принимает твои команды в свободной форме:**
- "добавь в план релиз трека на следующую пятницу"
- "перенеси пост с четверга на субботу"
- "вот фотки — добавь в очередь"

Агент обновляет контент-план в Obsidian через Syncthing — изменения появляются у тебя через секунды.

## Что агент НЕ делает (твои задачи)

- Публикация в Instagram (API закрыт)
- Финальный монтаж видео — творческий выбор за тобой
- Прослушивание и одобрение треков
- Живые съёмки и фотосессии
- Решения об образе и анонимности

## Как работает синхронизация

```
Obsidian на Mac  →→→  Syncthing  →→→  VPS (агент)
                                          ↓
                                     читает план
                                     обновляет план
                          ←←←  Syncthing  ←←←
Obsidian видит изменения (через ~2-5 сек)
```

## Команды бота

| Команда | Что делает |
|---------|------------|
| `/start` | Приветствие и список команд |
| `/plan` | Задачи на сегодня |
| `/week` | Обзор на неделю |
| `/queue` | Файлы готовые к публикации |
| `/generate` | Запустить генерацию контента (Phase 2) |
| Любой текст | Свободная команда агенту |
| Фото | Добавить в очередь |
